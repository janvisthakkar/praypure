# Praypure - Design Specification

## Brand Identity
- **Brand Name**: Praypure
- **Tagline**: "Purity Meets Prayer"
- **Industry**: Spiritual & Incense Products
- **Target Audience**: Spiritually inclined individuals, temple-goers, meditation practitioners

## Color Palette (From Logo)

### Primary Colors
- **Dark Brown**: #3E2723 / rgb(62, 39, 35) - Primary backgrounds, headers
- **Golden/Tan Accent**: #D4AF37 / rgb(212, 175, 55) - CTAs, highlights, premium elements
- **White**: #FFFFFF - Text on dark, borders, cards

### Extended Palette
- **Light Brown**: #5D4037 - Secondary backgrounds
- **Cream/Beige**: #F5F5DC - Alternate backgrounds, subtle sections
- **Warm Gold**: #F4A460 - Hover states, secondary accents
- **Charcoal**: #2C2C2C - Text on light backgrounds
- **Soft White**: #FFF8DC - Card backgrounds

## Typography

### Headings
- **Font Family**: 'Playfair Display', 'Cormorant Garamond', serif (elegant serif matching logo)
- **Sizes**: 
  - H1: 48px / 3rem
  - H2: 36px / 2.25rem
  - H3: 28px / 1.75rem
  - H4: 24px / 1.5rem

### Body Text
- **Font Family**: 'Inter', 'Poppins', sans-serif (clean, readable)
- **Size**: 16px (base), 14px (small), 18px (large)
- **Line Height**: 1.6

### Brand Font
- **Logo Style**: Elegant serif with prayer hands icon integration

## Spacing System
- **Base Unit**: 8px
- **Scale**: 8, 16, 24, 32, 40, 48, 64, 80, 96, 120

## Component Guidelines

### Buttons
- **Primary**: Golden background (#D4AF37), dark brown text, rounded (8px)
- **Secondary**: Transparent with golden border, golden text
- **Hover**: Darker gold (#B8941F), scale (1.02)
- **Padding**: 12px 32px

### Cards
- **Shadow**: 0 4px 12px rgba(62, 39, 35, 0.15)
- **Border Radius**: 12px
- **Background**: White or cream
- **Padding**: 24px

### Navigation
- **Height**: 80px
- **Background**: White with subtle shadow
- **Sticky**: Yes
- **Logo**: Prayer hands icon integration

## Layout Principles
- **Max Width**: 1200px (container)
- **Grid System**: 12-column grid
- **Gutters**: 24px
- **Responsive Breakpoints**: 
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+

## Brand Elements
- **Prayer Hands Icon**: Key visual element (Anjali Mudra)
- **Scalloped Borders**: Decorative element from logo
- **Elegant Serif Typography**: Maintains brand consistency
- **Golden Accents**: Premium feel, highlights important elements

## User Experience Principles
1. **Elegance**: Sophisticated, refined design matching brand
2. **Spirituality**: Peaceful, calming aesthetic
3. **Trust**: Quality indicators, testimonials
4. **Accessibility**: WCAG AA compliant, readable fonts
5. **Performance**: Optimized images, lazy loading
6. **Visual Hierarchy**: Clear product categorization
